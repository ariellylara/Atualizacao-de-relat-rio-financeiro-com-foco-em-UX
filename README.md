# Atualização de Relatório Financeiro com Foco na Experiência do Usuário (UX)

## Sobre o Projeto

Este projeto foi desenvolvido como parte da Formação Power BI Analyst.

O desafio consistiu em reformular um relatório financeiro existente, aplicando conceitos de UX (User Experience) para tornar a navegação mais intuitiva, melhorar a leitura das informações e destacar os indicadores mais relevantes para a tomada de decisão.

## Objetivos

- Melhorar a experiência do usuário no dashboard;
- Aplicar conceitos de posicionamento e hierarquia visual;
- Utilizar contraste para destacar informações importantes;
- Organizar os visuais de forma equilibrada;
- Implementar segmentações e filtros para análise dinâmica;
- Criar navegação entre páginas através de botões interativos.

## Ferramentas Utilizadas

- Power BI Desktop
- Power Query
- DAX
- Modelagem de Dados
- Design de Dashboards

## Indicadores Analisados

- Sales (Vendas)
- Profit (Lucro)
- Discounts (Descontos)
- COGS (Custo dos Produtos Vendidos)
- Units Sold (Unidades Vendidas)

## Melhorias Aplicadas

### Posicionamento
Reorganização dos elementos visuais para criar uma leitura mais natural e intuitiva.

### Contraste
Utilização de cores e elementos visuais para aumentar a legibilidade e destacar informações estratégicas.

### Proporção e Disposição
Distribuição equilibrada dos gráficos e indicadores, priorizando os dados mais importantes.

### Segmentação dos Dados
Implementação de filtros e segmentadores para análises por período, produto, país e segmento.

### Navegação
Criação de botões de navegação para melhorar a experiência do usuário entre as páginas do relatório.
